How metrics are calculated

Backlog by age counts only pending items using time since submitted_at.

Average TAT by payer uses records with both submitted_at and completed_at and status not pending. Payers are sorted by slowest average turnaround first.

Denial rate by payer is denied / decided, where decided = denied + verified/approved (pending excluded).

Top denial reasons shows the top 5 reasons by count (denied records only).
